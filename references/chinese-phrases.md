# Chinese Phrases to Trim or Replace

Initial Chinese reference list for common AI-sounding words in prose. Default move: cut first, then replace only if the sentence needs support.

## Overstated or Absolute Words

These push the tone too hard. Delete them or replace them with a more precise phrase.

| Avoid | Problem | Use instead |
|-------|---------|-------------|
| 最 | Tone too absolute | Delete or use a specific comparison |
| 最关键的 | Overstated emphasis | 更重要的 / 关键在于 |
| 最痛的点 | Packaged label | Name the actual problem |
| 只有 | False exclusivity | Delete if unnecessary |
| 全部 | Too absolute | 基本上都 |
| 甚至 | Manufactured escalation | Delete |
| 完全 | Empty intensifier | Delete |
| 根本 | Empty intensifier | Delete |
| 真正 | Empty emphasis | Delete |
| 任何 | Sweeping claim | Delete |
| 直接 | Empty emphasis | Delete |
| 超级 | Empty intensifier | Delete |

## Hard-Turn Fillers

These often create a fake reversal or an unnecessary setup.

| Avoid | Problem | Use instead |
|-------|---------|-------------|
| 其实 | Forced pivot | Delete |

## Empty Meta Phrases

These announce the structure instead of carrying the sentence.

| Avoid | Problem | Use instead |
|-------|---------|-------------|
| 一句话总结 | Empty framing | Delete |
| 也就是说 | Empty restatement | Delete |
| 把你接住 | Performative voice | Delete |

## Platformy or Packaged Words

These sound like content-ops language instead of clean prose.

| Avoid | Problem | Use instead |
|-------|---------|-------------|
| 照抄 | Undesired tone | 参考一下 |
| 补齐 | Feels mechanical | 查漏补缺 |
| 整合 | Vague packaging | 整理 |
| 爆点 | Overhyped label | Delete |
| 照样 | Weak filler | Delete |

## Editing Rule

If a sentence depends on one of these words to feel strong, the sentence probably needs a more specific noun, verb, or example instead.
