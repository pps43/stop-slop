# Chinese Structures to Avoid

Initial Chinese reference list for sentence patterns that make prose sound templated or AI-generated.

## Forced Reversal

These create a visible rhetorical turn before the actual point.

| Pattern | Problem | Use instead |
|---------|---------|-------------|
| `不是 X，而是 Y` | Hard pivot, over-signaled contrast | State `Y` directly |
| `不是 X，更像是 Y` | Forced reframing | Keep the more accurate image and drop the setup |

## Announced Summary

These label the sentence before delivering it.

| Pattern | Problem | Use instead |
|---------|---------|-------------|
| `一句话总结：...` | Empty preamble | Give the conclusion directly |
| `也就是说，...` | Repeats the same point | Delete the bridge and rewrite cleanly |

## Performative Setup

These try to create intimacy or force attention instead of earning it with content.

| Pattern | Problem | Use instead |
|---------|---------|-------------|
| `把你按住...` | Showy narrator voice | Start with the information |

## Editing Rule

If the first clause only exists to negate, summarize, or build suspense, cut that clause and test whether the sentence becomes cleaner.
